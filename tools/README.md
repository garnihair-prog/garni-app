# tools/

開発補助用のスクリプトです。アプリ本体の動作には不要です。

## make_qr_card.py

お客様向け予約ページのURLから、QRコードカード画像（PNG）を生成します。
店舗のPCやスマホにインストールして使うものではなく、**Claude（このアプリを作った開発環境）に「QRコードを作って」と頼むと、このスクリプトを使って画像を作成**します。

- `libqrencode`（システムライブラリ）を利用してQRコードを生成し、Pillowで店舗ロゴ・案内文とあわせてカード状に仕上げます。
- pip等のインストールが使えないサンドボックス環境でも動くように、外部パッケージ（`qrcode`等）を使わず、システムに入っている `libqrencode.so.4` をPythonの `ctypes` から直接呼び出しています。
- 使い方（Claudeに依頼する場合の例）：「デプロイ後のURLは https://garni-app-〇〇.onrender.com/ です。QRコードカードを作り直してください」と伝えるだけでOKです。

手元のパソコンで直接実行したい場合は、`libqrencode` とNoto Sans CJK JPフォントがインストールされたLinux環境が必要です（Windows環境では動作しません）。
